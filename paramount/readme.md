## Android client

~~~
com.cbs.app
~~~

Install system certificate.

## How to get aid?

In the response to the same request, you should see something like this:

~~~xml
<param name="trackingData" value="aid=2198311517|b=1000|bc=CBSI-NEW|ci=1|cid=1...
~~~
